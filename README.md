## 🚀 Pro Features & God-Tier Upgrades

- **💎 Glassmorphism HUD:** Translucent panels, animated status bars, and high-fidelity typography.
- **📈 Real-time Analytics:** Monitor FPS, particle counts, and render latency (ms) in a dedicated performance panel.
- **📜 Gesture Log:** A live history timeline of detected gestures for technical feedback.
- **🔥 Combo System:** Sustain powers to build a combo multiplier for enhanced immersion.
- **⚡ Vectorized Distortion:** High-speed `cv2.remap` based spatial warping for Water and Dark Matter powers.
- **⌨️ Direct Mapping:** Instant power selection using keys `1` through `7`.
- **🏗️ Clean Architecture:** Modular power classes, robust gesture detection, and a professional project structure.

## 🎮 Command Center (Controls)

### Hand Gestures
- **Open Palm:** Primary stream/distortion activation.
- **Fist (Hold 2.5s):** Quantum Charge. Release for a massive **BLAST**.
- **Index Point:** Precision Beams / Target Locking.
- **Pinch (Thumb+Index):** Telekinetic Object Grab.
- **Swipe (L/R):** Dynamic Power Cycling.
- **Both Hands Open:** Activate **ULTIMATE** Power mode.

### Keyboard Shortcuts
- `1` - `7`: Direct Power Selection.
- `H`: Toggle Cinematic Help Overlay.
- `R`: Toggle HD Video Recording.
- `S`: Instant Screenshot.
- `F`: Toggle Fullscreen Mode.
- `Q` / `ESC`: System Shutdown.
